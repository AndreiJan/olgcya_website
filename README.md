# OLGC YA Website 
This is a development of the Website for OLGCYA. 
Encompases the ministries handled and managed by OLGCYA and parterning ministries 


### Architecture 
The architecture of this set up is microservice. 
Services: 
- Receiver: Receives incoming data
- Storage: Communicates with our database
- Emailer: Sends emails to list of contacts in Storage service
